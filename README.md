## 1. Prerequisites

- numpy==1.21.2
- scipy==1.10.1
- matplotlib==3.5.3

```shell
pip3 install -r requirements.txt
```

## 2. Usage

- We provide a simple example for reference:

  ```shell
  python3 simple_example.py
  ```

- Real data experiment:

  ```shell
  python3 real_data_experiments.py
  ```
